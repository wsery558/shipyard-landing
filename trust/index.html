Trust Center

Shipyard is built for clarity and control. This page explains how we handle security, privacy, and responsible disclosure—without overpromising.

Security at a glance

  • Exportable evidence: activity logs and artifacts can be packaged for review.
  • Integrity support: checksums help detect tampering during handoff.
  • Least surprise: we prefer explicit outputs over vague assurances.

Privacy

This site is a product landing experience. If you submit the waitlist form, your responses are stored by the form provider. We also collect lightweight usage events (page views and clicks) to improve the site and the product.

  • What we collect: page path, timestamp, language, and basic referrer/UTM when present.
  • What we avoid: we do not ask for secrets, API keys, or private source code via this website.
  • Data sharing: we do not sell personal data.

Vulnerability Disclosure

We appreciate responsible reports. Please include steps to reproduce, impact, and any proof-of-concept that is safe to share. We aim to acknowledge reports and coordinate a fix, but we do not guarantee timelines.

  • Contact: see security.txt and SECURITY.md.
  • Scope: the landing site, public demos, and Shipyard-related repos.
  • Safe harbor: good-faith testing that avoids harm is welcome.

Release notes

Product and repo changes are tracked in GitHub commits/releases. When a public changelog exists, we link it here.

Contact

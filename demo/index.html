Common issues

Port already in use?

Shipyard defaults to :8080. Pass --port 9000 to use a different port.

pnpm/npm version?

We recommend pnpm 8+. If using npm, ensure npm install works.

Node.js version?

Requires Node 18+. Check: node --version.
